## NOTES GÉNÉRALES POUR HEYGEN

### Paramètres communs à tous les scripts
- **Langue** : Français
- **Avatar recommandé** : Homme ou femme africain(e) d'environ 25-40 ans, tenue décontractée mais propre (pas trop formal)
- **Fond** : Fond neutre ou légèrement coloré (vert émeraude #059669 possible en bannière)
- **Sous-titres** : Activer les sous-titres automatiques (accessibilité + TikTok les favorise)
- **Format vidéo** : 9:16 vertical (TikTok natif)
- **Musique de fond** : Volume bas (20-30%), musique afrobeat légère ou piano doux selon le script

### Ordre de publication recommandé
1. Script 3 (Démo) — pour tester la réaction
2. Script 4 (Témoignage) — pour construire la confiance
3. Script 1 (Douleur financière) — pour toucher la masse
4. Script 5 (Astuce business) — pour l'engagement et partages
5. Script 2 (Avant/Après) — pour convertir les intéressés
6. Script 6 (Appel émotionnel) — pour les retargeting / réchauffage

Toujours : **"Commente MARGO et je t'envoie le lien en DM"** — répondre manuellement pour l'instant, à automatiser via n8n.
